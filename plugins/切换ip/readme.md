# 切换静态IP插件 v1.1.0

在局域网中快速切换静态 IP 地址，支持阿里云 DDNS 自动更新。

## 功能

### IP 管理
- **向前/向后扫描**：在当前 IP 的同网段内向前（IP 递减）或向后（IP 递增）搜索空闲 IP
- **手动设置**：手动填入 IP 地址、子网掩码、默认网关，支持切换适配器
- **恢复 DHCP**：一键恢复为自动获取 IP，保留静态 DNS 配置
- **查看网络信息**：显示所有网络适配器的详细配置

### 阿里云 DDNS
- **DDNS 配置**：设置阿里云 AccessKey、域名、主机记录等参数
- **自动更新**：定时检测公网 IP 变化，自动更新 DNS 解析记录
- **手动更新**：一键将当前公网 IP 同步到阿里云 DNS
- **状态查看**：查看当前 DNS 记录、公网 IP、更新历史

> ⚠ 修改 IP 需要管理员权限，请以管理员身份运行主程序。
